title: Echo
category: "ext"
---

The `echo` command applies coloring, borders, and spacing to text.

### Examples {#echo-examples}

```shell
$ heimdall echo --foreground '#FFFFFF' --border line --padding "1 4" Error
```
```text
┌─────────────┐
│             │
│    Error    │
│             │
└─────────────┘
```
